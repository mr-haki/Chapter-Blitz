# 🏴‍☠️ ChapterBlitz

ChapterBlitz is a manga hub built with HTML, CSS, and JavaScript, created to help manga fans explore chapter summaries, information, reviews, and discussions in one simple platform.

## ✨ Features

- 📚 Manga information pages
- 📖 Chapter summary pages
- 🔍 Manga search recommendations
- ⭐ Planned rating system
- 📝 Planned review features
- 🎨 Custom responsive design

## 🛠️ Built With

- HTML
- CSS
- JavaScript

v0.2
